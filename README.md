# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Muse'
Import-Module -Name 'Muse'
```

To install privately:

```powershell
Save-Module -Name 'Muse' -Path '.'
Import-Module -Name '.\Muse'
```
